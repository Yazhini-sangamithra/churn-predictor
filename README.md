# churn-predictor

## Overview
This is a Machine Learning web application that predicts whether a customer will churn (leave the service) based on their data.  
The app is built using **Python, Scikit-learn, and Streamlit** and provides real-time predictions through an interactive UI.


## Problem Statement
Customer churn is a major issue in subscription-based businesses.  
This project helps identify customers who are likely to leave so that companies can take preventive actions to improve retention.

## Machine Learning Model
- Algorithm: Logistic Regression / Random Forest (mention yours)
- Data preprocessing: Label encoding, scaling
- Model saved using: Joblib / Pickle
- Output: Churn (Yes/No)
